GALERIE-BILDER
==============

Lege hier deine Arbeiten als Bilddateien ab, z. B.:
  hund-spaziergang.jpg
  portraet-licht.jpg
  layout-plakat.png

Danach in der Datei  projekte.js  einen Eintrag ergänzen, z. B.:

  {
    bild: "assets/galerie/hund-spaziergang.jpg",
    titel: "Mein Begleiter",
    kategorie: "tiere",
    text: "Tierfotografie"
  },

Empfehlung: Bilder vorher auf ca. 1200 px Breite verkleinern
(spart Ladezeit). JPG für Fotos, PNG für Grafiken.
